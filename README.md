# An Gia Green Mobile

Expo SDK 54 React Native app scaffolded from the An Gia Green Figma flow.

## Run

```bash
npm install
npm run start
```

Open with Expo Go compatible with SDK 54.

## Figma assets

Replace the placeholder PNG files in `src/assets` and `src/assets/icons` with exported assets from the Figma file:

- `logo.png`
- `splash.png`
- `icon.png`
- `adaptive-icon.png`
- `icons/icon-home.png`
- `icons/icon-products.png`
- `icons/icon-chat.png`
- `icons/icon-notifications.png`
- `icons/icon-profile.png`

Keep these filenames so components continue using the designed logo and icons.

npx.cmd expo start --clear
